# 三、每个功能的核心实现逻辑（直接可开发）

## 1）代码审查智能体
**输入**：代码片段/文件/仓库地址
**AI 提示词核心**：
```
你是资深代码审查专家，专注Java/Spring/前端。
请审查以下代码，从：
1. 语法错误
2. 安全漏洞
3. 性能问题
4. 编码规范
5. 可读性
给出审查报告，并标出问题行。
```
**输出**：结构化 JSON（方便前端渲染）

---

## 2）改进建议智能体
和代码审查联动，**直接输出可落地优化方案**：
- 优化后代码
- 为什么这么改
- 替代方案（如用线程池、缓存、异步等）

---

## 3）审批智能体（最实用）
流程：
1. 代码审查通过 → 自动创建审批单
2. 推送至企业微信/钉钉
3. 审批人：同意 / 驳回 / 加意见
4. 结果回写 Agent → 自动通知提交人

你如果不想 heavy 用 Flowable，我可以给你**轻量审批表结构**。

---

## 4）聊天智能体
就是标准 LLM 对话，**支持上下文记忆**：
- 问技术问题
- 让 AI 直接帮你审查代码（聊天里粘贴即可）
- 查审批进度
- 查学习资料

---

## 5）自助学习智能体（RAG）
- 上传：开发规范、部署文档、常见问题、架构文档
- AI 自动切片 → 向量化 → 存入向量库
- 用户提问 → 检索最相关内容 → AI 回答
