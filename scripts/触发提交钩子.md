已创建两种集成方案：

## 使用方法

### 方案1: 本地Git Hook (推荐)

```bash
cd your-project
bash scripts/install-hook.sh
```

每次 `git commit` 时自动检查代码：
- 严重问题 → 阻止提交
- 非严重问题 → 警告但允许提交

### 方案2: GitLab Webhook

1. 在GitLab项目设置 → Webhooks
2. 添加URL: `http://your-server/api/code-review/webhook/gitlab`
3. 选择 "Push events" 触发

### 支持语言
- Java
- Python  
- JavaScript/Vue/TypeScript
- Go
