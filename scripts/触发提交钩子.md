已创建两种集成方案：

## 使用方法

### 方案1: 本地Git Hook (推荐)

```bash
cd your-project
bash scripts/install-hook.sh
```

每次 `git commit` 时自动检查代码：
- 严重问题 → 阻止提交
- 非严重问题 → 警告但允许提交

### 方案2: GitLab Webhook

1. 在GitLab项目设置 → Webhooks
2. 添加URL: `http://your-server/api/code-review/webhook/gitlab`
3. 选择 "Push events" 触发

### 支持语言
- Java
- Python  
- JavaScript/Vue/TypeScript
- Go

---

## Windows 安装使用方法

### 方法1: 一键安装
```bash
cd your-project
运行 scripts\install-hook-windows.bat
```

### 方法2: 手动安装
```powershell
# 复制脚本到 hooks 目录
copy scripts\pre-commit.ps1 .git\hooks\pre-commit

# 修改执行策略（如果需要）
Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy RemoteSigned
```

### 功能特性
- ✅ 自动检查暂存的所有代码文件
- ✅ 支持Java、Python、JavaScript、Vue、Go等
- ✅ 严重问题(红色) → 阻止提交
- ✅ 重要问题(黄色) → 警告但允许
- ✅ 彩色终端输出
